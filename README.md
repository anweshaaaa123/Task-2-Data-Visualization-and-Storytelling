# Super Store Sales Dashboard (Power BI)

## Overview
This Power BI dashboard provides an interactive analysis of Super Store sales data, enabling users to monitor sales performance, profitability, customer trends, and regional performance. It also includes a 15-day sales forecasting page for future sales analysis.

## Features
- Interactive dashboard with slicers for filtering data by **Region**.
- KPI cards displaying:
  - Total Sales
  - Total Profit
  - Total Quantity Sold
  - Average Delivery Time
- Sales analysis by:
  - Category
  - Region
  - State
  - Payment Mode
- Geographic visualization of sales distribution using a map.
- Monthly and category-wise sales trend analysis.
- **15-Day Sales Forecast** using Power BI forecasting.
- State-wise sales comparison with interactive visuals.

## Tools & Technologies
- **Power BI Desktop**
- Power Query
- DAX Measures
- Interactive Charts & Maps
- Forecast Analytics

## Dashboard Pages
### Page 1 – Sales Overview
- Business KPIs
- Category-wise Sales
- Region-wise Sales
- Payment Mode Distribution
- Geographic Sales Map
- Interactive Region Filter

### Page 2 – Sales Forecast
- Sales Trend Over Time
- 15-Day Sales Forecast
- State-wise Sales Performance

## Outcome
The dashboard helps users analyze historical sales performance, identify business trends, compare regional performance, and make data-driven decisions using interactive visualizations and forecasting.
